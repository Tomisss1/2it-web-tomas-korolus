# Osobní One-Page vizitka

Moderní jednostránkový web vytvořený jako školní projekt do soutěže 2IT WEB.

## Použité technologie
- HTML5
- CSS3
- JavaScript

## Funkce
- Responzivní design
- Animace při scrollu
- Moderní UI


## Minecraft WEB Project
Spuštění
Stačí otevřít soubor index2.html v prohlížeči.

minecraft-field-guide/
│
├── index.html              # Hlavní stránka (Rozcestník / Biomy)
├── monsters.html           # Stránka s monstry (Moby)
├── o-projektu.html         # Stránka splňující AI integraci a info
├── robots.txt              # Pro SEO (vyžadováno zadáním)
├── sitemap.xml             # Pro SEO (vyžadováno zadáním)
├── README.md               # Kompletní dokumentace (hlavní část odevzdání)
│
├── css/
│   ├── style.css           # Globální styly, reset, typografie
│   ├── layout.css          # Flexbox / Grid rozvržení stránek
│   └── components.css      # Styly pro karty biomů/monster, tlačítka
│
├── js/
│   ├── main.js             # Společný JS (např. mobilní menu)
│   └── filter.js           # JS pro interaktivní filtrování monster
│
└── assets/
    ├── images/             # Optimalizované .webp obrázky monster a biomů
    │   ├── biomes/
    │   └── monsters/
    └── icons/              # Ikony (např. srdíčka pro životy, meč pro útok)
